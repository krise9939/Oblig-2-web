GitHub

Git og GitHub er en fildelingsservice. Det brukes for dele filer mellom flere personer, så flere kan gjøre endringer i de samme dokumentene. Dette gjør samarbeid enklere. I GitHub er det også funksjoner som lagrer et dokkument hver gang det blir gjort endringer. Dette gjør at man enkelt kan laste en gammel versjon av dokumentet enkelt, om det skulle skje noe med dokumentet, eller om man må reversre enkelte deler av teksten til en versjon som ellers ville ha gått tapt.
